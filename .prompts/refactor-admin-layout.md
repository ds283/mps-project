## PREAMBLE

This task is designed to redesign the user interface for the main database functions that create "reference" data.

Specifically, under the existing "Site management..." menu, these are:

- Project classes
- Affiliations
- Degree programmes
- Presentation venues
- Asset licenses
- Role editor
- Global email templates

Also, under the "Admin..." menu, these are:

- Transferable skills
- Supervisory team
- Project tags

The difference between these two menus is that the "Admin..." functions are intended to be available to convenors,
so they can add items in these categories if needed. The other categories need an "admin" or "root" role.

### THE PROBLEM

This layout does not scale. As further reference items are added to the database, we need more entries on the
drop-down menu. But there is only limited space.

## TASK

Research and plan an alternative layout for these user interface functions.
